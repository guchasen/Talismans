# Talismans Mod - 符箓模组

一个为Minecraft添加道家风格符箓系统的Fabric模组，让玩家能够制作和使用各种具有神奇效果的符箓。

## 核心功能

### 1. 符箓制作系统
- **制符台**：专门用于制作符箓的工作台
- **制符工具**：制符笔和符纸作为基础材料
- **配方系统**：通过配方系统制作各种功能的符箓

### 2. 当前已有的符箓分类与效果

#### 增益符箓（Buff Talismans）
长按右键使用，为玩家提供持续性的增益效果：

- **增力符**：力量效果（600秒，等级2）
- **奇力符**：力量效果（1200秒，等级5）
- **神力符**：力量效果（1200秒，等级7）
- **增速符**：速度效果（300秒，等级1）
- **强跃符**：跳跃提升效果（200秒，等级1）
- **奇运符**：幸运效果（1800秒，等级1）
- **神运符**：幸运效果（1800秒，等级3）
- **强抗符**：抗性提升效果（600秒，等级1）
- **抗火符**：火焰抗性效果（1200秒，等级1）
- **水神符**：水下呼吸效果（600秒，等级1）
- **夜明符**：夜视效果（3600秒，等级1）
- **匿形符**：隐身效果（1200秒，等级1）
- **回春符**：生命恢复效果（600秒，等级1）
- **续气符**：瞬间治疗效果

#### 投掷（Throwable Talismans）
投掷使用，产生各种攻击性效果：

- **落雷符**：召唤闪电攻击
- **怒雷符**：增强版雷电符箓，在落地位置数次召唤闪电
- **雷暴符**：在落地点一定范围内多次召唤雷电
- **明火符**：在落地点一定范围内生成火焰
- **起爆符**：爆炸攻击符箓
- **灵爆符**：强力爆炸符箓


### 基础使用
1. **获取材料**：首先需要获得制符笔和符纸
2. **制作符箓**：在制符台中使用配方制作符箓
3. **使用符箓**：
   - 增益符箓：长按右键激活
   - 投掷符箓：右键投掷使用


## 版本信息

- **当前版本**：0.5.0
- **Minecraft版本**：1.20.1+
- **模组加载器**：Fabric
